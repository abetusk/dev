Caucy Convergence Test

$$ S = \sum_{k=0}^{\infty} a_k < \infty \ \iff \ \forall \epsilon > 0, \exists N \in \mathbb{N} \ \forall n > N, p \ge 0 \ \ (n,p \in \mathbb{N}) \ | \sum_{s=0}^{p} a_{n+s} | < \epsilon $$
